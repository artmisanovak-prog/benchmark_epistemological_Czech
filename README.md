# benchmark_epistemological_Czech
# Mouse: Epistemological Observation Benchmark
## Součást projektu "Chceš myš?? [slovník mezer]"

### O BENCHMARKU
Tento benchmark testuje schopnost AI pracovat s:
1. **Epistemologickým pozorováním** - pozorování bez předem daných interpretačních rámců
2. **Sémiotikou mezer** - práce s prázdny a mezerami jako nositeli významu
3. **Meta-jazykovými otázkami** - rozpoznání, kdy jazyk mluví o sobě samém
4. **Symbolickou transformací** - chápání, jak se entity mění v nástroje významu

### STRUKTURA
- `fairy_tale.txt` - Kompletní text pohádky "O myši"
- `illustration.jpg` - Vizuální komponenta testu
- `benchmark_manifest.json` - Hlavní metadata
- `implicit_rules.json` - Implicitní pravidla kognitivního systému pohádky
- `cognitive_tasks.json` - 5 testovacích úkolů
- `evaluation_criteria.json` - Kritéria hodnocení
- `visual_annotation.json` - Analýza ilustrace pro multimodální test

### JAK POUŽÍVAT
1. **Pro testování AI**: Dejte AI přečíst pohádku, ukázat ilustraci a zadat úkoly z `cognitive_tasks.json`
2. **Pro výzkum**: Analyzujte, jak AI/pokročilé modely pracují s epistemologickou nejednoznačností
3. **Pro výuku**: Ukázka komplexní literární analýzy s filozofickým přesahem

### TESTOVANÉ SCHOPNOSTI
1. **Fenomenologická akceptace** - přijímání jevů tak, jak jsou
2. **Práce s mezerami** - využívání prázdna jako významového prostoru
3. **Meta-lingvistické vědomí** - rozpoznání jazyka o jazyce
4. **Symbolická fluidita** - chápání proměnlivosti významu

### OČEKÁVANÁ SELHÁNÍ AI
- Bude se snažit anomálie vysvětlit vědecky
- Přehlédne meta-vrstvu otázky "Chceš myš?"
- Aplikuje naučené pohádkové šablony
- Bude preferovat jednoznačnou interpretaci
